# Knowledge Graph для материаловедения

Система автоматического извлечения знаний из научных статей с построением графа связей и семантическим поиском.

##  Что делает система

- **Парсит PDF-статьи** и извлекает текст
- **Находит сущности** (организации, люди, места, материалы) через NLP
- ️ **Строит граф знаний** в Neo4j
- **Создаёт эмбеддинги** для семантического поиска
- **Визуализирует** связи в интерактивном формате
- **Отвечает на вопросы** по содержимому статей
